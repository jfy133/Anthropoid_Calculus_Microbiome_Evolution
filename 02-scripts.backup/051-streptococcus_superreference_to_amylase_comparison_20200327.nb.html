Get total Streptococcus super-reference reads.

## Collapse the contigs
superreference_data_meta <- superreference_data %>% 
  group_by(genus, sample, species) %>%
  summarise(total_no_reads = sum(no_reads),
             total_no_bases_covered = sum(no_bases_covered),
             total_species_length = sum(feature_length),
             mean_mean_depth_coverage = mean(mean_depth_coverage)
             ) %>%
    mutate(percent_covered = (total_no_bases_covered / total_species_length) * 100) %>%
  group_by(sample) %>%
  mutate(total_dataset_reads = sum(total_no_reads)) %>%
  left_join(raw_metadata_deep %>% 
              select(sample_name, Env, Description, Host_General, 
                     Host_Genus, Host_Common, Age, total_unmapped_reads), 
            by = c("sample" = "sample_name")
            ) %>%
    filter(Host_Genus != "Control")

superreference_data_total <- superreference_data_meta %>%
  filter(genus == "Streptococcus") %>%
  group_by(sample) %>%
  summarise(total_genus_reads = sum(total_no_reads)) %>%
  rename(individual = sample)

Firstly we can make a reduced data tibble.

However, we don’t have normally distributed data so many the G-A plots aren’t the best.

Instead, we can perform a Mann-Whitney U test as a non-parametric test that sampling between two independent populations will likely result as being greater or lower between the two populations.

We match the assumptiosn that: observations between the groups are independent, we have orddinal data, null hypothesis is the distributions are equal, and alternate is not-equal.

To do this in R:

## Base R
wilcox.test(data = data_reduced$`abpA-like`, `Ratio of abp-associated reads to total Streptococcus reads` ~ Group) %>% broom::tidy()
cannot compute exact p-value with ties
wilcox.test(data = data_reduced$`abpB-like`, `Ratio of abp-associated reads to total Streptococcus reads` ~ Group) %>% broom::tidy()

These are relatively well supported results, as a Mann-whitney U test interally does a form of permutation (if I understand it correctly), as it essentially looks how many times each sample in one group is higher than each sampele in the other. In both cases we have large statistic values and p -values of much less than an alpha 0.05

We should also check the effect size, and the precision of the estimation. Here we use the statistic r (which is the Z-value divded by number of sample points), and then a confidence interval (at 95%)

# With Coin
rcompanion::wilcoxonR( data_reduced$`abpA-like`$`Ratio of abp-associated reads to total Streptococcus reads`,  data_reduced$`abpA-like`$Group, ci = T)
rcompanion::wilcoxonR( data_reduced$`abpB-like`$`Ratio of abp-associated reads to total Streptococcus reads`,  data_reduced$`abpB-like`$Group, ci = T)
